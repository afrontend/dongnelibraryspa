# DongneLibrary SPA

도서관 책을 빌릴 수 있는지 확인하는 웹 서비스

## Using npm
```
git clone https://github.com/afrontend/dongnelibraryspa
cd dongnelibraryspa
npm ci
npm start
```
`http://localhost:3000/` 주소에 접속하여 도서 검색

## Using Web

무료 서버에서 동작함으로 약 10초의 로딩 시간이 걸릴 수 있다.
http://dongne.herokuapp.com/
