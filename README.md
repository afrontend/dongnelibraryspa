## 도서관 책이 대출되었는지 확인하는 웹서비스

### 설치
```sh
$ git clone https://github.com/afrontend/dongnelibraryspa
$ cd dongnelibraryspa
$ npm install
```

### 사용 예
```sh
$ npm start
```
